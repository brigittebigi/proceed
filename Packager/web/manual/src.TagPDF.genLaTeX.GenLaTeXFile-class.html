set_header_style(self, style)

source code 

Fix the header style.

Parameters:
  • style, (string)., style, is, one, of -
    • \rm - Roman
    • \it - Italics
    • \em - Emphasis (toggles between \it and m)
    • \emph (LaTeX2e emphasis command)
    • \bf - Boldface
    • \sl - Slanted
    • \sf - Sans serif
    • \sc - Small caps
    • \tt - Typewriter
