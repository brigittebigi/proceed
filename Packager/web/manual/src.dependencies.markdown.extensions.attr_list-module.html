Attribute List Extension for Python-Markdown

Adds attribute list syntax. Inspired by [maruku](http://maruku.rubyforge.org/proposal.html#attribute_lists)'s feature of the same name.

See <https://pythonhosted.org/Markdown/extensions/attr_list.html> for documentation.
