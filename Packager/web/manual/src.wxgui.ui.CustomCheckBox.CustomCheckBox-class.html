Class CustomCheckBox

source code

wx.PyControl --+
               |
              CustomCheckBox

A custom class that replicates some of the functionalities of wx.CheckBox, while being completely owner-drawn with a nice check bitmaps.

(Constructor)

source code 

Default class constructor.

Parameters:
  • parent - Parent window. Must not be None.
  • id - CustomCheckBox identifier. A value of -1 indicates a default value.
  • label - Text to be displayed next to the checkbox.
  • pos - CustomCheckBox position. If the position (-1, -1) is specified then a default position is chosen.
  • size - CustomCheckBox size. If the default size (-1, -1) is specified then a default size is chosen.
  • style - not used in this demo, CustomCheckBox has only 2 state
  • validator - Window validator.
  • name - Window name.
