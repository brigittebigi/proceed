  CreateSession
Used to create Session instances.

The Session ID is controlled as follow:

  • starts by '['
  • ends by ']'
  • first letter is uppercase
  • first letter is:
    • K for keynotes
    • O for oral session
    • P for poster session
    • S for demo
    • T for tutorial
    • anything else for other types of sessions

The first letter of the session name is used while sorting by session types (in the writers).
