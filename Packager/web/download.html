Download

Proceed is ready to run, so it does not need elaborate installation, except for its dependencies (other software required for Proceed to work properly).

All you need to do is to copy the Proceed package from the web site to somewhere on your computer. Preferably, choose a location without spaces nor accentuated characters in the name of the path.

The package is compressed and zipped, so you will need to decompress and unpack it once you've got it.
